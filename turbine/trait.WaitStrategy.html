fn wait_for(&self, sequence: u64, ep: &Vec<&Padded64>) -> u64

Wait for the requested sequence, but return the largest available

Provided a target cursor position and a slice of dependency cursors, this method will block until a slot is available to read from. The method of blocking varies depending on the implementation (e.g. busy-wait, sleep, etc).

This method should return the highest available position in the buffer to allow EventProcessors to batch reads
